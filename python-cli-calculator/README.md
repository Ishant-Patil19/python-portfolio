# 🧮 Python CLI Calculator

A simple and beginner-friendly **command-line calculator** built using Python.
This project demonstrates core programming concepts like functions, loops, dictionaries, and user input handling.

---

## 🚀 Features

* Perform basic arithmetic operations:

  * ➕ Addition
  * ➖ Subtraction
  * ✖️ Multiplication
  * ➗ Division
* Continuous calculations (chain operations)
* Clean and interactive CLI interface
* Beginner-friendly and easy to understand

---

## 🛠️ Tech Stack

* **Python 3.x**
