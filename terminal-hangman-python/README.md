🎮 Terminal Hangman (Python)


> A clean and beginner-friendly Hangman game built using Python with ASCII visuals and modular design.


🚀 Why This Project?

Most beginner Python projects are messy and unstructured.

This project solves that by:

* ✅ Demonstrating **clean modular coding**
* ✅ Showcasing **game logic fundamentals**
* ✅ Providing a **portfolio-ready CLI project**


🧩 Features

* 🎯 Random word selection
* ❤️ Lives system with ASCII visuals
* 🔁 Input validation
* 🧠 Tracks guessed letters
* 🧱 Modular code structure


🕹️ How to Play

1. Run the game
2. Guess one letter at a time
3. Each wrong guess reduces lives
4. Win by guessing the full word before lives run out
