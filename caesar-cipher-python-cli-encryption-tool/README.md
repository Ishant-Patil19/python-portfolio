# 🔐 Caesar Cipher Python Tool

A simple and beginner-friendly **Caesar Cipher encryption & decryption tool built using Python**. This CLI-based project allows users to encode and decode messages using shift-based substitution.

---

## 🚀 Features

* Encode (encrypt) messages
* Decode (decrypt) messages
* Handles non-alphabet characters (spaces, symbols, numbers)
* Beginner-friendly Python logic
* Interactive command-line interface

---

## 🧠 What is Caesar Cipher?

The **Caesar Cipher** is one of the oldest encryption techniques. It works by shifting letters in the alphabet by a fixed number.

Example:

* Input: `hello`
* Shift: `3`
* Output: `khoor`

---

## 🛠️ Tech Stack

* Python 3
* CLI (Command Line Interface)

---

## 💻 Usage

```bash
Type 'encode' to encrypt, type 'decode' to decrypt:
encode
Type your message:
hello
Type the shift number:
3
```

Output:

```
khoor

```
